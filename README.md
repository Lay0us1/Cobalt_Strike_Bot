# Cobalt_Strike_Bot
CobaltStrike 上线通知，飞书群聊机器人通知

## 飞书群聊机器人
在飞书中新建一个群聊，在群聊中添加一个群聊机器人。将其webhook地址复制，替换`$feishu_url`中的webhook地址
